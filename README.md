## Tech Stack
- Kotlin
- AndroidX
- Rx
- Retrofit / OkHttp
- Gson
- Groupie as Recyclerview lib
- sdp ssp
- Koin as Dependency Injection lib
- Picasso as Image loader

## Extra
- `./gradlew detekt` static analysis with detekt
- `./gradlew ktlint` kotlin linter
- `./gradlew check` for running all
